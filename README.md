# Project Title - Test

Final project for the Building AI course

## Summary

The project consists in analysing and summarising the information contained in a technical FaceBook group.

## Background

Some FaceBook groups contains a lot of valuable information shared among the users over time. This information is diluted within out of scope or non pertinent comments. The information is not structured and repeated over time because users tends to post new questions without to look for previous similar posts. The project will analyses posts and build a kind of user manual presenting the information in a structured manner. The project is applicable to any technical FaceBook group. Users of the project will immediately access the information contained in the group without wasting time in searching posts or reading non pertinent information. Users will continue to contribute to their group with new topics not yet discussed.

## How is it used?

The process will consist in running the solution on a dedicated FaceBook group in order to get in a structured manner like a user anual the information issed from answers to the various questions raised within the group over time. The process can be reiterated from time to time so new infomration can enter the user manual and existing information can be modified or updated with recent post information..

## Data sources and AI methods
The data comes from the various posts of the group. Additionnally, the manufacturer user manuals, if existing, can represent an additonal source of informatioon. The data is directly collected by the solution.

## Challenges

One challenge will consist in evaluating the truth of some answers especially when very few others answers bring a confirming  information.
The limitation comes from the information provided which depends upon a specific context applicable only to the user who posted the information, making the provided information not necessarily useful for others users.

## What next?

The solution can expand first by looking at different groups discussing the same subject within FaceBook, and then by looking at multiple platforms dealing with the same subject.
